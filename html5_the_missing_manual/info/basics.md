`Ways to include css.`
```html
<head>
    <!--    in head link-->
    <link href="styles.css" rel="stylesheet">
    <!--in head style tag-->
    <style>
        p {
            color: red;
            font-size: 14px;
        }
    </style>
</head>
<body>
<!--in style attribute-->
<p style="color: red; font-size: 14px;"> it's a paragraph </p>
</body>
```
